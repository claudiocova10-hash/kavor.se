# Kavor Brand Kit 1.0

## Färger
- Primär röd: #E30613
- Mörk grafit: #0D1117
- Stålgrå: #6B7280
- Vit: #FFFFFF

## Filer
- `kavor-primary-dark.svg` – primär logotyp med slogan på mörk bakgrund.
- `kavor-horizontal-light.svg` – horisontell logotyp för ljus bakgrund.
- `kavor-icon-dark.svg` – ikon för mörk bakgrund/appikon.
- `kavor-icon-light.svg` – ikon för ljus bakgrund.
- `kavor-primary-dark.png` – PNG-version av primär logotyp.
- `kavor-icon-1024-dark.png` – appikon 1024x1024.
- `favicon.ico` – favicon för hemsidan.

## Rekommenderad användning
- Hemsida header: `kavor-horizontal-light.svg` eller `kavor-primary-dark.png`.
- Appikon: `kavor-icon-1024-dark.png`.
- Favicon: `favicon.ico`.
- Mörk bakgrund i app/hemsida: `kavor-primary-dark.svg`.
